What it does

⚙

Auto-detects everything

OS, architecture, Node.js version, existing OpenClaw install. Installs what's missing, skips what's already there.

⚡

Validates before saving

Telegram bot tokens and AI API keys are checked against live APIs before they touch your config. No broken setups.

📄

Generates your agent files

SOUL.md, USER.md, and HEARTBEAT.md are built from your answers. Your agent knows who you are from minute one.

🔌

Starts on boot

Configures launchd (macOS) or systemd (Linux) so your agent is always running. No manual restart needed.

🩺

Built-in diagnostics

oc-setup doctor checks everything and tells you exactly what's wrong and how to fix it.

🔄

Idempotent & safe

Run it again if something fails. Every step checks current state first. Reset backs up before wiping.

Commands

npx oc-setup

Full guided setup. Detects OS, installs OpenClaw, configures Telegram, sets up your AI model, generates personality files, starts the gateway, and verifies everything works.

oc-setup doctor

Diagnoses common issues: Node version, port conflicts, expired tokens, gateway crashes. Outputs a health report with one-liner fix commands.

oc-setup update

Updates OpenClaw to latest, preserves your config, restarts the gateway automatically.

oc-setup reset

Nuclear option. Backs up current config to a timestamped folder, wipes everything, and re-runs setup from scratch.
