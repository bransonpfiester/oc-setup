WhatsApp setup happens during onboarding.

  1. Run the setup command below
  2. During onboarding, select WhatsApp as your channel
  3. Scan the QR code with your phone (WhatsApp > Linked Devices > Link a Device)
Note: WhatsApp may need re-linking every ~14 days.
  1. Go to Discord Developer Portal
  2. Create a New Application, then go to Bot > Reset Token
  3. Copy the token and paste it above
  4. Under OAuth2 > URL Generator, select "bot" scope, then invite it to your server

Signal setup happens during onboarding.

  1. Run the setup command below
  2. During onboarding, select Signal as your channel
  3. The wizard installs signal-cli and guides you through account linking
Requires Java 21 for signal-cli.

iMessage works on macOS only. Two options:

  1. Native (legacy): Uses the macOS Messages database directly
  2. BlueBubbles (recommended): Install BlueBubbles server on your Mac, then enter the server URL during onboarding

What oc-setup handles for you

Detects everything

OS, architecture, Node.js, existing installs. Skips what's there, installs what's missing.

Validates before saving

Telegram tokens and API keys are checked against live APIs before touching your config.

Generates your agent files

SOUL.md, USER.md, HEARTBEAT.md built from your preferences. Your agent knows you from minute one.

Auto-starts on boot

Configures launchd (macOS) or systemd (Linux). Your agent is always running.

Built-in diagnostics

oc-setup doctor checks everything and tells you exactly how to fix it.

Safe to re-run

Every step is idempotent. If something fails, run it again. Reset backs up before wiping.

Commands

npx oc-setup

Full guided setup. Detects OS, installs OpenClaw, configures Telegram, sets up your AI model, generates personality files, starts the gateway, verifies everything.

oc-setup doctor

Diagnoses common issues: Node version, port conflicts, expired tokens, gateway crashes. Outputs a health report with fix commands.

oc-setup update

Updates OpenClaw to latest, preserves your config, restarts the gateway.

oc-setup reset

Backs up current config, wipes everything, re-runs setup from scratch.
